Searches for the value of key. Returns array of matches. Each match has the object at last level, last key and value of the whole key. The key can have "*" to search in all elements of array or all keys in an object.

Parameters:

  • obj Object | Array

    Object/Array to search in.

  • key String

    A deep key. Can have "" at all positions except last. Eg: a.b.c, a..b

  • [insert] Boolean optional

    To insert keys if not present or not. Defaults to false.

  • [expandKeys] Array optional

    Array of key maps for each * of objects in the key to use intead of all keys. Defaults to all.

Returns:

Array:

Array of matches. Each match is an array [object at last level, last key, value of the whole key]. Returns null if no matches are found.
